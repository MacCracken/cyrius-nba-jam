# Contributing to cyrius-nba-jam

See [AGNOS First-Party Standards](https://github.com/MacCracken/agnosticos/blob/main/docs/development/applications/first-party-standards.md).

## Key Rules

- All code is new Cyrius — the original TMS34010 source is a DESIGN REFERENCE, not code to translate
- Study the original system (DRONE.ASM, BB.ASM, etc.) before implementing, then write fresh
- Do not distribute NBA-licensed content — assets are user-provided
- Frame time benchmarks required for rendering claims
- Play test every change — feel matters as much as correctness
